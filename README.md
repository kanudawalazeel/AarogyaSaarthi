# AarogyaSaarthi
AarogyaSaarthi is a modern, role-based healthcare web application designed to improve coordination between patients, doctors, pharmacies, and administrators. The platform provides dedicated dashboards for each role, enabling efficient appointment management and access control. Built using React and TypeScript, the application follows a modular and scalable frontend architecture. The UI is optimized for clarity, usability, and real-world healthcare workflows. This project was developed as part of a hackathon to demonstrate practical digital healthcare solutions.
